# Resham Chaudhary

Cybersecurity technologist. Founder of [Kumari.ai](https://kumari.ai). Based in Chicago.

I work on the unglamorous parts of software — the network paths a packet actually takes to reach a service, the etcd quorum that keeps a Kubernetes control plane alive, the WireGuard mesh that lets my homelab talk to a Hetzner box like they're on the same wire. Most of what's here started as a problem I needed to solve in my own infrastructure and ended up as a write-up at [reshamchaudhary.com](https://reshamchaudhary.com).

Currently building **Kumari.ai** — an intelligent chat platform with multi-agent reasoning and routing across LLMs. Before that, three years of running a production-grade homelab on three Dell OptiPlex 7050s and a managed switch.
